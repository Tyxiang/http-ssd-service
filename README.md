# HTTP Object

## 1. 概述 Overview

这是一个作为服务存在的对象，您可以用 RESTful 方式配置它的属性和方法，使其成为您需要的任何服务。 

This is an object that exists as a service. You can configure its properties and methods in a RESTful way to make it any service you need.
